# eomcs-java-web-project
이곳은 [엄진영의 코딩스쿨]에서 강의하는 자바 웹 애플리케이션 프로젝트의 소스 파일을 보관하는 저장소이다. 단계별로 프로젝트를 진행하면서 자바 웹 프로그래밍의 기술을 익힐 수 있도록 `step01`, `step02` 식으로 프로젝트가 구분되어 있다.  

## 대상자
- 자바 기본문법을 학습한 사람
- 서블릿/JSP를 학습하였거나 학습하려는 사람
- 스프링 프레임워크 기반 프로젝트에 참여중이거나 참여할 예정인 사람
- 자바 웹애플리케이션 프로젝트의 유지보수를 맡고 있거나 맡을 예정인 사람

## 목표
- 자바 웹 애플리케이션 개발 방법을 배우고 익히기
- 웹 애플리케이션 개발에 사용되는 다양한 기술의 적용 방법을 배우기
- 과거에서 최근까지 웹 애플리케이션의 구조가 변화되고 발전되는 과정을 경험하기
- 다양한 구조의 웹 애플리케이션을 유지보수 할 수 있는 능력을 갖추기

## 강의 주제
각 단계의 프로젝트 강의 주제는 다음과 같다.

- 목표
    - 애플리케이션 아키텍처 종류와 각 아키텍처 별 특징을 이해한다.
- 내용
    - PC 애플리케이션
    - C/S 애플리케이션
    - 웹 애플리케이션
    - 모바일 웹 애플리케이션
    - 모바일 앱

#### 자바 웹 애플리케이션 아키텍처
- 목표
    - 자바 웹 애플리케이션 아키텍처와 구동 원리를 이해한다.
- 내용
    - 웹 컴포넌트
    - 요청 처리 절차

#### 자바 웹 애플리케이션 프로젝트 준비
- 목표
    - 웹 애플리케이션 프로젝트를 위한 개발 도구를 준비하고 설정할 수 있다.
    - 글로벌 표준 웹 프로젝트 폴더를 생성할 수 있다.
    - 웹 애플리케이션을 서버에 배포하여 실행할 수 있다.
- 내용
    - Gradle 빌드 도구 준비
    - 프로젝트 폴더 생성
    - HTML 페이지 작성
    - 웹 모듈 생성
    - 서블릿 컨테이너 준비
    - 웹 모듈 배치 및 실행
- 구현
    - HTML로 웹 애플리케이션 시작 페이지를 만든다.

#### 이클립스 IDE 사용
- 목표

    - 이클립스 IDE로 웹 애플리케이션 프로젝트 다룰 수 있다.
- 내용
    - 이클립스 IDE
    - Gradle용 Eclipse 플러그인
    - 프로젝트 임포트
    - 서블릿 컨테이너 등록
    - 개발 테스트용 서버 환경 구축
    - 웹 애플리케이션 배치와 실행

### 서블릿
#### 서블릿 만들기
- 목표
    - 서블릿을 만들 수 있다.
    - 서블릿의 생성과 실행, 소멸 과정을 이해한다.
- 내용
    - Servlet 인터페이스
    - 서블릿 생명주기
    - 배치 설명서(web.xml)
    - @WebServlet 애노테이션
    - 서블릿 실행
- 구현
    - '회원 목록', '팀 목록', '게시물 목록' 서블릿을 만든다.
    - 시작 화면의 메뉴에 이들 서블릿을 연결한다.

#### 서블릿을 만드는 더 쉬운 방법
- 목표
    - GenericServlet 추상 클래스의 용도를 이해한다.
    - GenericServlet을 이용하여 서블릿을 만들 수 있다.
- 내용
    - GenericServlet 추상 클래스와 Servlet 인터페이스의 관계
    - GenericServlet 추상 클래스 상속과 service() 추상 메서드
- 구현
    - '회원 목록', '팀 목록', '게시물 목록' 서블릿을 GenericServlet의 서브 클래스로 만든다.

#### HTTP 요청/응답 프로토콜
- 목표
    - HTTP 요청과 응답 프로토콜을 이해한다.
- 내용
    - HTTP 요청 프로토콜
    - HTTP 응답 프로토콜
    - ServletRequest 와 ServletResponse 객체

#### 웹 브라우저로 출력하기
- 목표
    - 웹 브라우저로 콘텐츠를 출력할 수 있다.
- 내용
    - HTTP 응답과 ServletResponse 객체
    - 문자열 출력하기
    - 한글이 깨지는 이유
    - HTTP 응답 헤더 'Content-Type'
- 구현
    - '회원 목록', '팀 목록', '게시물 목록'을 요청했을 때 각 페이지의 제목을 출력한다.

#### 데이터베이스에서 데이터 가져오기
- 목표
    - DBMS 서버에서 데이터를 가져와 웹 브라우저로 출력할 수 있다.
- 내용
    - DBMS 서버 설치와 설정
    - 테이블 생성과 예제 데이터 입력
    - JDBC 드라이버 준비와 배치
    - JDBC 프로그래밍
    - 웹 브라우저로 데이터 출력
- 구현
    - '회원 목록', '팀 목록', '게시물 목록' 데이터를 가져와 웹 브라우저로 출력한다.

#### 웹 페이지 UI 꾸미기
- 목표
    - HTML과 CSS의 용도를 이해한다.
    - HTML의 주요 태그를 다룰 수 있다.
    - CSS를 이용하여 태그의 모양을 다룰 수 있다.
    - HTML과 CSS를 이용하여 웹 페이지를 꾸밀 수 있다.
- 내용
    - HTML 요점 정리
    - CSS 요점 정리
    - 화면 작성에 HTML 적용하기
    - HTML에 CSS 적용하기
- 구현
    - HTML과 CSS를 사용하여 '회원 목록', '팀 목록', '게시물 목록' 화면을 꾸민다.

#### 웹 브라우저에서 웹 서버로 데이터 보내기
- 목표
    - GET과 POST 요청 프로토콜의 특징을 이해한다.
    - GET과 POST 요청을 이용하여 웹 브라우저에서 웹 서버로 데이터를 보낼 수 있다.
- 내용
    - GET과 POST 요청의 특징 및 장단점
    - GET 요청으로 데이터를 보내는 방법
    - POST 요청으로 데이터를 보내는 방법
- 구현
    - GET 요청을 이용하여 '회원 목록', '팀 목록', '게시물 목록' 서블릿을 요청한다.
    - GET 요청을 이용하여 '회원 상세정보', '팀 상세정보', '게시물 상세정보' 서블릿을 요청한다.
    - POST 요청을 이용하여 '회원 등록', '팀 등록', '게시물 등록' 서블릿을 요청한다.
    - POST 요청을 이용하여 '로그인' 서블릿을 요청한다.

#### HTTP 요청에서 데이터 꺼내기
- 목표
    - 웹 브라우저가 보낸 HTTP 요청에서 데이터를 추출할 수 있다.
- 내용
    - GET과 POST 요청과 ServletRequest 객체
    - HTTP 요청에서 데이터 추출하기
    - 한글 데이터가 깨지는 이유
- 구현
    - '회원 상세정보', '팀 상세정보', '게시물 상세정보' 서블릿을 만든다.
    - '회원 등록', '팀 등록', '게시물 등록' 서블릿을 만든다.
    - '로그인' 서블릿을 만든다.
